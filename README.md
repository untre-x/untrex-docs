# UntreX Documentation

Welcome to the official documentation repository for **UntreX**.

UntreX is a modern storytelling platform where people can create, organize, and preserve meaningful life stories with rich media, privacy controls, and a clean reading experience.

## Documentation

This repository contains developer documentation for UntreX integrations and platform guides.

### Available Guides

- Supabase Integration
- Authentication
- Database Structure
- Storage
- Security Best Practices

## Supabase Integration

UntreX uses Supabase to power:

- Authentication
- PostgreSQL Database
- Row Level Security (RLS)
- Storage
- Realtime Features

Read the complete guide here:
